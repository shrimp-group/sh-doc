# 文件服务

Micro-File 是文件管理微服务模块，提供文件上传、下载、存储管理功能。支持阿里云 OSS、AWS S3、MinIO 等多种存储后端，具备灵活的 Bucket 配置管理和文件访问签名机制。

## 功能特性

- **多存储支持**：支持阿里云 OSS、AWS S3、MinIO 等主流对象存储
- **Bucket 管理**：支持多 Bucket 配置，支持默认 Bucket 自动选择
- **访问控制**：支持私有读（签名访问）和公有读两种访问模式
- **文件类型限制**：支持图片、视频文件大小和格式限制
- **缓存机制**：Bucket 配置本地缓存 + Redis 分布式同步
- **多租户支持**：支持按租户隔离 Bucket 配置

## 适用场景

- 用户头像上传
- 文档附件管理
- 图片资源存储
- 视频文件托管
- 富文本编辑器文件上传

## 技术栈

- Java 25
- Spring Boot
- MyBatis
- Redis
- 阿里云 OSS SDK
- AWS S3 SDK
