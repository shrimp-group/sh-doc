# 字典服务

Micro-Dict 是字典数据微服务模块，提供系统字典的统一管理。支持字典类型和字典项的增删改查，具备本地缓存和分布式同步机制，为业务系统提供高效、稳定的字典数据服务。

## 功能特性

- **字典类型管理**：支持字典分类和字典类型的统一管理
- **字典项管理**：支持字典项的增删改查，支持批量操作
- **本地缓存**：字典数据本地缓存，减少数据库查询
- **分布式同步**：通过 Redis 实现多实例缓存同步
- **动态刷新**：支持字典数据动态刷新，无需重启服务
- **复制粘贴**：支持字典数据的复制和粘贴，便于环境迁移

## 适用场景

- 用户状态（正常、禁用、锁定）
- 订单状态（待支付、已支付、已发货、已完成）
- 数据字典（性别、民族、地区）
- 业务类型（订单类型、支付方式）
- 系统配置（开关配置、参数配置）

## 技术栈

- Java 25
- Spring Boot
- MyBatis
- Redis
