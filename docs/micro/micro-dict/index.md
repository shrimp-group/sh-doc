# 字典服务

Micro-Dict 是字典数据微服务模块，提供系统字典的统一管理功能。支持字典的增删改查、缓存管理、多租户隔离等特性。

## 功能特性

- **字典管理**：支持字典类型和字典项的完整管理
- **缓存支持**：字典数据本地缓存，提高访问性能
- **动态刷新**：支持字典配置的动态刷新
- **多租户支持**：支持按租户隔离字典数据
- **REST API**：提供完整的 REST 接口
- **常用接口**：提供常用的字典查询接口

## 适用场景

- 系统配置项管理
- 业务状态码定义
- 下拉选择数据
- 枚举值管理
- 系统参数配置

## 技术栈

- Java 25
- Spring Boot
- MyBatis
- Redis
